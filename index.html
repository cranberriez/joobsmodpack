Installation Guide

Curseforge Installation

  • Step 1 - Downloading CurseForge

    If you don't already have the CurseForge launcher installed, you can download it here.

    You will probably want to download the Standalone version to avoid Overwolf bloat.

  • Step 2 - Downloading The Pack

    Download the modpack either here or via the download button above.

  • Step 3 - Creating Profile

    Navigate to the minecraft tab of cursefoge and select Create Custom Profile.

  • Step 4 - Importing

    Select import and find the .zip you downloaded.

  • Step 5 - Increasing Minecraft's Memory

    Navigate to Profile Options and modify the amount of memory your game will be using.

    Adjust the memory according to the memory available on your system.

    • System
      Game
    • 8 GB
      5,500MB
    • 16 GB
      8,000MB
    • 32 GB
      12,000MB

    These are just recommended. Don't go below 5,500MB or above 12,000MB

Alternative Launchers

    Alternatives to CurseForge are not limited to these two examples, there are many others. Almost all alternatives have methods to import CurseForge packs.

    Importing the modpack should be a similar process to Curseforge with buttons in different places. Don't forget to adjust memory.

    These launchers might automatically open a browser during the installation process. This is NORMAL, as some mods are required to be downloaded this way due to third party limitations.

  • Prism Launcher

    Prism Launcher can be downloaded here. It is an adfree alternative to Curseforge, although basic looking it has more functionality than CurseForge. It is safe.

  • AT Launcher

    AT Launcher can be downloaded here. Another alternative to CurseForge and is very similar to Prism Launcher. Slightly harder to navigate than Prism or Curse.

  • Other Options

    GD Launcher is another good and safe alternative and can be found here


    MultiMC is a lesser known but good alternative as well. I cannot personally vouch for this launcher though. Found here

Note About Safety

About a month ago an individual gained access to multiple Curse Forge Mod accounts and uploaded malicious version of those mods to be available to the public.

This was very quickly discovered and taken down. Mod developers and communities quickly released fixes and tools to detect the malicious code which locked down the problem.

This was scary but was fixed quickly. I have done my due diligence when creating this pack. All mods have updated and checked to confirm no such malicious code.

The modpack will rarely be updated, and all mod updates will be double checked to confirm validity.
