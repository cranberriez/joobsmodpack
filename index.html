Installation Guide

CurseForge Installation

  • Step 1 - Downloading CurseForge

    If you don't already have the CurseForge launcher installed, you can download it on the CurseForge Launcher page.

    You will probably want to download the Standalone version to avoid Overwolf bloat.

  • Step 2 - Downloading The Pack

    Download the Lite Modpack here or via the download button above. You don't have to unzip it.

  • Step 3 - Creating Profile

    Navigate to the minecraft tab of CurseForge and select Create Custom Profile.

  • Step 4 - Importing

    Select import and find the .zip you downloaded. You don't have to unzip it.

  • Step 5 - Increasing Minecraft's Memory

    Navigate to Profile Options and modify the amount of memory your game will be using. You can also modify this in CurseForge settings in the minecraft tab to make this a global setting.

    Adjust the memory according to the memory available on your system.

    • System
      Game
    • 8 GB
      5,500MB
    • 16 GB
      8,000MB
    • 32 GB
      12,000MB

    These are just recommended. Don't go below 5,500MB or above 12,000MB

Note About Safety

A couple months ago an individual gained access to multiple Curse Forge Mod accounts and uploaded malicious code version of those mods to be available to the public. It was referred to as fractureiser.

This was very quickly discovered and taken down. Mod developers and communities quickly released fixes and tools to detect the malicious code which locked down the problem.

This was scary but was fixed quickly. I have done my due diligence when creating this pack. All mods have updated and checked to confirm no such malicious code.

The modpack will rarely be updated, and all mod updates will be double checked to confirm validity.
